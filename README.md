# Lejurobot Challenge

## Setup

Create conda environment:
```bash
conda create -n lejurobot python=3.11
conda activate lejurobot
```

Install package:
```bash
pip install .
```

## Data

Download the dataset from:
https://huggingface.co/datasets/LejuRobotics/kuavo_data_challenge_icra
